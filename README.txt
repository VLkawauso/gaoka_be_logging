使い方
code → download zipでダウンロード。
zipを展開(winだと解凍?)
gaokaファイルを適当な場所に保存。
PCを適切なWi-Fiに繋ぐ。
ターミナルを起動。
gaokaファイルのあるディレクトリに移動。
node gaoka.js 101　(最後の数字は部屋番号に置き換え)を実行。
→ファイル取得用のURLがデフォのブラウザで開く。
node gaoka_sc.js 101　(最後の数字は部屋番号に置き換え)を実行。
→スクショ取得用のURLがデフォのブラウザで開く。
